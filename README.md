# 💰 AI Smart Finance Assistant with Scam Detection

## 🚀 Features
- Add Income & Expenses
- Real-time Balance Calculation
- Prevent Overspending (Low Balance Alert)
- Transaction History
- Scam Detection (UPI/Fake Messages)

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB Atlas

## 🔥 Highlights
- Full Stack Project
- Real-world problem solving
- Secure transaction validation
- Clean UI Dashboard

## ▶️ Run Locally
1. Clone repo
2. Run backend:
   node server.js
3. Open frontend with Live Server
